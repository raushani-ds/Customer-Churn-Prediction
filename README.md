# Customer Churn Prediction

## Project Overview

Customer Churn Prediction is a Machine Learning project that predicts whether a telecom customer is likely to leave the company or continue using its services.

The project uses the Telco Customer Churn dataset and applies data preprocessing, exploratory data analysis (EDA), feature engineering, and machine learning models to identify churn patterns.

---

## Objectives

- Analyze customer behavior and service usage.
- Identify factors affecting customer churn.
- Build predictive machine learning models.
- Compare model performance and select the best model.

---

## Dataset

Dataset: Telco Customer Churn Dataset

Features include:

- Gender
- Senior Citizen
- Partner
- Dependents
- Tenure
- Phone Service
- Internet Service
- Monthly Charges
- Total Charges
- Contract Type
- Payment Method
- Churn

---

## Machine Learning Models

### Logistic Regression

Accuracy: 78.94%

### Random Forest Classifier

Accuracy: 79.32%

Best Model: Random Forest Classifier

---

## Key Findings

The most important factors affecting customer churn are:

- Tenure
- Total Charges
- Monthly Charges
- Internet Service Type
- Contract Type

Customers with longer tenure and long-term contracts are less likely to churn.

---

## Results

- Logistic Regression Accuracy: 78.94%
- Random Forest Accuracy: 79.32%

---

## Conclusion

The Random Forest model performed better than Logistic Regression and achieved approximately 79.3% accuracy.

This project demonstrates how machine learning can help telecom companies identify customers at risk of leaving and improve retention strategies.
